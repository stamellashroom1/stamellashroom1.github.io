Maths Tools

Here are some tools I have made. All locally running code!

General Calculator

You can use this to solve basic operations. Only use the characters 0-9 and +, -, *, /, ., and ^. It will follow order of operations. Working on support for brackets.


Line of Best Fit

Input datapoints and this calculator will produce a linear line of best fit. Enter data in the format x1, y1, x2, y2 etc. Data needs to be ordered by x value.



Mean Calculator

Enter values in a comma-separated format and the calculator will find the mean.


Derivative Finder - Coming Soon!

Enter an expression and the calculator will find its first derivation.
